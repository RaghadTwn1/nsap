# 💛 نِصاب | Nisab

منصة ذكية تساعد المستخدمين على إدارة مقتنياتهم الذهبية، وحساب زكاة الذهب تلقائيًا، مع تقديم مساعد ذكي يعتمد على الفتاوى الموثوقة.

---

## 📌 فكرة المشروع

يهدف مشروع نِصاب إلى تسهيل حساب زكاة الذهب من خلال منصة إلكترونية تجمع جميع بيانات الذهب في مكان واحد، وتتابع اكتمال الحول، وتحسب مقدار الزكاة المستحقة، مع توفير مساعد ذكي للإجابة عن الأسئلة الشرعية المتعلقة بزكاة الذهب.

---

## ✨ المميزات

- 📱 تسجيل الدخول وإنشاء حساب.
- 🪙 إدارة جميع مشتريات الذهب.
- 🧮 حاسبة زكاة الذهب.
- 🔔 تنبيهات عند اكتمال الحول.
- 🤖 مساعد ذكي للإجابة عن الأسئلة الشرعية.
- 📚 أسئلة شائعة.
- ☎️ التواصل مع المفتي.
- 🏪 عرض المحلات المرتبطة.
- 👤 صفحة معلومات الحساب.
